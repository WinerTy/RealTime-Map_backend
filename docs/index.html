Документация Socket.IO API

Здесь описаны все события (events), которые используются для real-time взаимодействия с нашим сервером.

Подключение

URL для подключения: wss://your-backend-domain.com

События, отправляемые клиентом (Client → Server)

Событие: sendMessage

Отправляет новое сообщение в чат.

Данные (payload): 
{
  "room": "string",
  "text": "string"
}
Ответ сервера (acknowledgement): 
{
  "status": "ok",
  "messageId": "string"
}

События, получаемые клиентом (Server → Client)

Событие: newMessage

Сервер присылает это событие всем участникам комнаты при появлении нового сообщения.

Данные (payload): 
{
  "fromUser": "string",
  "text": "string",
  "timestamp": "iso_string_date"
}
